# Basic Sample Hardhat Project

This project is a NFT collection built on NEXT JS using typescript.

## Deploy Contract
$ npx hardhat run scripts/deploy.js --network mumbai

## compile
$ npx hardhat compile

## Deploy
$ npx hardhat run scripts/deploy.js --network mainnet
$ npx hardhat run scripts/deploy.js --network mumbai

## Test
$ npx hardhat test
